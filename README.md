# Project Name: BookNest

A seamless and modern web application designed to digitize the traditional library experience. Users can explore a vast collection of books, filter by categories, and borrow titles digitally.

## Live URL

[https://booknest.vercel.app](https://my-eighth-assignment-9gtf.vercel.app/)

## Purpose

BookNest aims to bring the traditional library experience into the digital age. Users can register, log in, browse books, filter by category, and borrow books — all from one platform.

## Key Features

- Secure Authentication with BetterAuth (Email/Password & Google OAuth)
- Browse a full collection of books
- Search books by title
- Filter books by category (Story, Tech, Science)
- View detailed information about each book
- Borrow books with a single click
- Personal profile page with update functionality
- Fully responsive on mobile, tablet, and desktop
- Unique and modern UI with DaisyUI and Tailwind CSS
- Smooth animations with Animate.css
- Scrolling marquee for new arrivals

## Tech Stack

- **Framework:** Next.js
- **Styling:** Tailwind CSS, DaisyUI
- **Authentication:** BetterAuth
- **Database:** MongoDB Atlas
- **Language:** JavaScript (React)

## Description

BookNest is a modern and fully responsive online book borrowing platform built to digitize the traditional library experience. Designed with simplicity and performance in mind, BookNest allows users to explore a rich collection of books, filter them by categories such as Story, Tech, and Science, and borrow titles digitally — all from one seamless interface.
The platform features a secure authentication system powered by BetterAuth, supporting both Email/Password login and Google OAuth, ensuring a safe and smooth sign-in experience. All user data and book records are stored and managed using MongoDB Atlas, providing a reliable and scalable cloud database solution.
Built with Next.js and styled using Tailwind CSS and DaisyUI, BookNest delivers a clean, intuitive, and visually appealing user interface that works beautifully across mobile, tablet, and desktop devices. Users can create and manage their personal profiles, update their information, and enjoy a personalized library experience.
Whether you are a casual reader looking for your next favorite book or a tech enthusiast searching for programming resources, BookNest is your go-to digital library — available anytime, anywhere.
